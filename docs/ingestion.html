02 / V2 Data Fetchers INGESTION PIPELINE

The V2 Ingestion Pipeline represents a massive expansion over the legacy system, integrating global patents, open-source repositories, and clinical trials alongside traditional academic papers.

The Guaranteed Quota Algorithm

In V1, generic queries could monopolize the user's digest (e.g., an overwhelming number of AI papers crowding out CRISPR patents). The V2 engine implements a strict Quota Guarantee: If a subscriber selects 5 different domains, Noetica ensures at least one discovery from each selected domain is included before filling the remaining slots by raw significance score.

Expanded Source Registry

The source registry (`src/source_registry.py`) maps user domains to specific API endpoints and advanced search queries.

1. arXiv (Academic Literature)

Pulls from the global preprint server. Filters out low-quality preprints using basic structural validation on the abstract.

2. CrossRef (Clinical & Patents)

Leverages CrossRef's massive database to identify newly published clinical trials and registered patents across multiple disciplines (Oncology, Neuroscience, Robotics).

3. GitHub (Open Source Implementation)

Extracts trending repositories matching the subscriber's technical domains, ensuring that theoretical papers are paired with their practical code implementations.
